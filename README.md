# Modern HTML & CSS From The Beginning 2.0

Learning repository for the Modern HTML & CSS From The Beginning 2.0 course.

## Course Structure

- **01-essential-html** - HTML fundamentals and semantic elements
- **02-forms-and-inputs** - Form elements and input types
- **03-more-html-elements** - Audio, video, tables, SVG, and more
- **04-css-basics** - Selectors, fonts, colors, and styling
- **05-box-model-and-positioning** - Layout fundamentals
- **06-flexbox** - Flexible box layout
- **07-responsive-media-queries** - Responsive design techniques
- **08-various-css-features** - Custom properties, filters, and advanced features
- **09-web-accessibility** - ARIA attributes and accessibility testing
- **10-advanced-selectors** - Pseudo-elements and advanced CSS selectors
- **11-css-grid** - Grid layout system
- **12-transitions-animations-js** - Animations and JavaScript integration
